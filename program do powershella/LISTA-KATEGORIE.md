# Lista kategorii Windows 11 Cleaner

## Jezyk programu

Program wykrywa jezyk z ustawien Windows:

- `en` - angielski
- `de` - niemiecki
- pozostale lub nieznane - polski

W menu glownym opcja `5` pozwala zmienic jezyk recznie.

Potwierdzenia dzialaja lokalnie i uniwersalnie:

- polski: `TAK` albo `T`
- angielski: `YES` albo `Y`
- niemiecki: `JA` albo `J`

## Bezpieczne do usuniecia

Te rzeczy zwykle mozna usunac bez psucia systemu:

- Poczta / Outlook
- Microsoft Store
- Sticky Notes
- Solitaire Collection
- Pogoda
- Mapy
- Microsoft 365 / Office hub
- Microsoft To Do
- Feedback Hub
- Get Help
- Tips

## Opcjonalne

Usuwaj tylko, jesli wiesz, ze tego nie uzywasz:

- OneDrive - jesli nie synchronizujesz plikow z chmura
- Windows Update - blokada na wybrana liczbe dni albo stale wylaczenie z opcja przywrocenia
- Microsoft Edge - tylko jesli masz inna przegladarke
- Word
- Excel
- PowerPoint
- Outlook
- Outlook classic - zwykle jest czescia pakietu Office i moze nie usunac sie osobno
- OneNote
- OneNote for Windows 10
- Microsoft Copilot
- Copilot w Windows
- Xbox / Game Bar - jesli nie grasz i nie nagrywasz gier
- Teams / Chat - jesli nie uzywasz
- Widgets - jesli przeszkadzaja

## Nie ruszac

Tych rzeczy lepiej nie usuwac ani nie wylaczac na stale:

- Windows Update
- Windows Security / Defender
- Microsoft Edge WebView2 Runtime
- Visual C++ Redistributable
- .NET Runtime
- Sterowniki
- Windows Installer
- Uslugi aktywacji Windows
